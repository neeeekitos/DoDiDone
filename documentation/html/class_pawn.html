◆ firstMove

bool Pawn::firstMove

This variable tells wether the pawn is in the first move status (initial status). If the pawn is in its first move status, it can move two squares forward. Else, it can only move one square

◆ inPassing

bool Pawn::inPassing

This variable determines if the current pawn can do an "en passant" move. It is eventually set by another pawn making its FIRST MOVE and landing beside the current pawn
